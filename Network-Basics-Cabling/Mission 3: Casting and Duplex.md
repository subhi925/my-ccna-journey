Real-world mapping:

Unicast: A private chat or a direct voice call between two people on WhatsApp (1-to-1). (הערה: מכשיר קשר הוא דוגמה ל-Half-Duplex, אבל פה חיפשנו דוגמה לשידור מאחד לאחד).

Multicast: Sending a message or media inside a WhatsApp Group (1-to-many / specific group).

Broadcast: DHCP (When a device connects to the network and shouts to everyone to get an IP address). (הערה: ב-Zoom משתמשים בדרך כלל ב-Multicast או Unicast מול שרת, בעוד ש-DHCP היא דוגמת ה-Broadcast המושלמת).

Half-Duplex Slowdown Explanation:

"It will be very slow because a Half-Duplex interface cannot send and receive data at the same time. It can only do one thing at a time—either send or receive. When uploading files to a cloud application like Google Drive, the computer needs to upload data blocks and simultaneously receive acknowledgements (ACKs) from the server. In Half-Duplex, this causes collisions and forces the network card to wait, which severely slows down the upload speed."
